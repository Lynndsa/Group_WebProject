STEPHEN KING FAN PAGE
=====================

Информационный веб-сайт о легендарном американском писателе Стивене Кинге

О ПРОЕКТЕ
---------
Stephen King Fan Page — это информационное веб-приложение, посвящённое творчеству и жизни «Короля ужасов» Стивена Кинга. Сайт предоставляет пользователям возможность ознакомиться с биографией писателя, его библиографией, экранизациями и последними новостями.

ОСОБЕННОСТИ
-----------
- Биография: подробная информация о жизни Стивена Кинга
- Книги: каталог произведений автора
- Новости: актуальные новости о писателе
- Фотогалерея: коллекция фотографий разных лет
- Адаптивный дизайн: корректное отображение на всех устройствах
- Тёмная тема: стильный дизайн в цветах #0a0a0a и #bb0a0a

ТЕХНОЛОГИИ
----------
Backend:
  - Python 3.8+
  - Bottle 0.12+

Frontend:
  - HTML5
  - CSS3
  - JavaScript (ES6+)
  - Google Fonts (Playfair Display, Inter, Courier Prime)


ФУНКЦИОНАЛ
----------
Страница      URL       Описание
-----------   -------   ----------------------------------
Главная       /         Приветственный экран с информацией
Биография     /bio      Подробная биография и фотогалерея
Книги         /books    Каталог произведений Стивена Кинга
Новости       /news     Последние новости о писателе

ДИЗАЙН
------
Цветовая палитра:
  - Основной фон:     #0a0a0a (почти чёрный)
  - Фон карточек:     #1a1a1a (тёмно-серый)
  - Акцентный цвет:   #bb0a0a (красный)
  - Основной текст:   #f0f0f0 (белый дым)
  - Вторичный текст:  #a0a0a0 (серый)

Шрифты:
  - Playfair Display  — заголовки (с засечками)
  - Inter             — основной текст (без засечек)
  - Courier Prime     — моноширинный (даты, код)
